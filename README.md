# Student Performance Predictor

A full-stack machine learning application to predict student pass/fail status and final scores.

## Features

- **ML Models**: Logistic Regression + Random Forest comparison
- **FastAPI Backend**: RESTful API for predictions
- **React Frontend**: Interactive UI for student data input
- **Model Evaluation**: Accuracy, F1-Score, ROC-AUC metrics
- **Docker Support**: Easy deployment

## Tech Stack

- Backend: Python, FastAPI, scikit-learn
- Frontend: React, CSS3
- ML: pandas, numpy, scikit-learn
